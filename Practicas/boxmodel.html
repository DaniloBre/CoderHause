Caja y posiciones

  • Relativ: coloca el elemento segun la conveniencia del flujo normal del documento. Este puede superponerce sbre otro elemento
  • Absolute: Es removido de su flujo normal, este es recomendado establecerlo dentro de un relative, sino se establecera en el top de la pagina
  • Fixed: es para dejar pegado un elemento deonde le es indicado y que se desplace segun el limte que se le impone.
  • Sticky: forma ivrida entre el relaive y el fixed, se pega sobre el borde superior.
  • Z-index: es para cuando dos elemtos se superpponen este los ordena.
